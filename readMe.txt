#приложение под линукс
Проводник:

	Программа (remote_explorer) начинает работу в корневом каталоге всегда
	Работает управление мышью и клавиатурой (аналогично mc)

	Функционал:

		1) вывод информации о файлах на панелях аналогично mc
		2) переключение между панелями на Tab либо кликом мыши на элемент
		3) непосредственно открытие файлов и обход каталогов (двойным кликом мыши либо Enter)
		4) F1 (либо клик на соответствующую кнопку) - вывод информации о работе программы
		5) F2 (либо клик на соответствующую кнопку) - копирование
		6) F3 (либо клик на соответствующую кнопку) - перемещение
		8) F4 (либо клик на соответствующую кнопку) - удаление
		9) F5 (либо клик на соответствующую кнопку) - создание католога
		10) F6 (либо клик на соответствующую кнопку) - вывод прав доступа активного элемента
		11) F7 (либо клик на соответствующую кнопку) - переименование
		12) F8 (либо клик на соответствующую кнопку) - загрузка файла в левую панель из сервера
		13) F9 (либо клик на соответствующую кнопку) - загрузка файла в правую панель из сервера
		14) F10 (либо клик на соответствующую кнопку) - завершение программы
		15) добавлен стиль приложения.
	
	Для запуска:

    1) прописываем qmake ./remote_explorer (qmake сгенерирует makefile)
    2) далее выполняются обычные действия с make
    
Сервер:

	Программа (server) является севером для передачи файлов из каталога resources.
	Порт и адрес задается константой
	Пусть с файлами для перадачи задается константой (каталог уже создан,
	но есть возможность создать свой и поменять путь)
	Сервер работает только на передачу файлов
	
	Функционал:
	
		1) При помещении файла в каталог resouces клиенту можно загрузить его себе
		2) Ввод q завершает работу сервера
